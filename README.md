# ClassicLootManager

## About
Classic Loot Manager is intended to be a robust Loot Tracking and awarding tool for guilds. 
CLM Currently supports:
* DKP  

There are also some planned systems:
* EPGP
* Roll based
* Suicide Kings

AddOn is designed to superseed Monolith and its derivatives. It solves the biggest painpoints and is focused on optimial communication and storage. CLM implements EventSourcing and MVC programming models to achieve those targets. All data is synchronised automatically in very optimal way.
